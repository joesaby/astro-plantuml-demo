# Astro PlantUML Demo

A simple demo showcasing the astro-plantuml integration in a plain Astro project.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 🧑‍💻 Features

This demo includes examples of:
- Sequence diagrams
- Class diagrams
- Activity diagrams
- Use case diagrams
- State diagrams
- Component diagrams

## 📦 Dependencies

- [Astro](https://astro.build/)
- [astro-plantuml](https://www.npmjs.com/package/astro-plantuml)
